**«Blog Customizer»** — это учебный проект в рамках курса «Фронтенд-разработчик» в Яндекс Практикуме. Интерактивное веб-приложение, которое даёт возможность настраивать внешний вид блога через интерфейс. Пользователь может изменять стилистические параметры, а изменения применяются к контенту мгновенно после подтверждения действий. 

Для запуска Storybook выполните:

```
npm run storybook
```

Для запуска линтера для стилей выполните:

```
npm run stylelint
```

Для запуска линтера выполните:

```
npm run lint
```

Для запуска форматтера выполните:

```
npm run format
```
